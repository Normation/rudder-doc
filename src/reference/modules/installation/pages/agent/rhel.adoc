* on Amazon Linux:

----

echo '[Rudder_6.2]
name=Rudder 6.2
baseurl=http://repository.rudder.io/rpm/6.2/RHEL_6/
gpgcheck=1
gpgkey=https://repository.rudder.io/rpm/rudder_rpm_key.pub' > /etc/yum.repos.d/rudder.repo

----

* on Amazon Linux 2:

----

echo '[Rudder_6.2]
name=Rudder 6.2
baseurl=http://repository.rudder.io/rpm/6.2/RHEL_7/
gpgcheck=1
gpgkey=https://repository.rudder.io/rpm/rudder_rpm_key.pub' > /etc/yum.repos.d/rudder.repo

----

Install the package:

----

yum install rudder-agent

----
