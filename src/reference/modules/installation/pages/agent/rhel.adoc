include::{partialsdir}/rhel_repo.adoc[]

* on Amazon Linux:

----

echo '[Rudder_7.3]
name=Rudder 7.3
baseurl=http://repository.rudder.io/rpm/7.3/RHEL_6/
gpgcheck=1
gpgkey=https://repository.rudder.io/rpm/rudder_rpm_key.pub' > /etc/yum.repos.d/rudder.repo

----

* on Amazon Linux 2:

----

echo '[Rudder_7.3]
name=Rudder 7.3
baseurl=http://repository.rudder.io/rpm/7.3/RHEL_7/
gpgcheck=1
gpgkey=https://repository.rudder.io/rpm/rudder_rpm_key.pub' > /etc/yum.repos.d/rudder.repo

----

Install the package:

----

yum install rudder-agent

----
