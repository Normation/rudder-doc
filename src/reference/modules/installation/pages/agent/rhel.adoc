* on Amazon Linux 2023:

----

echo '[Rudder_9.0]
name=Rudder 9.0
baseurl=http://repository.rudder.io/rpm/9.0/AL_2023/
gpgcheck=1
gpgkey=https://repository.rudder.io/rudder_release_key.pub' > /etc/yum.repos.d/rudder.repo

----

[NOTE]
====

Downloading the Amazon Linux 1 & 2 agents requires an active subscription.

====

* on Amazon Linux 1:

----

echo '[Rudder_9.0]
name=Rudder 9.0
username=LOGIN
password=PASSWORD
baseurl=http://download.rudder.io/rpm/9.0/RHEL_6/
gpgcheck=1
gpgkey=https://repository.rudder.io/rudder_release_key.pub' > /etc/yum.repos.d/rudder.repo

----

* on Amazon Linux 2:

----

echo '[Rudder_9.0]
name=Rudder 9.0
username=LOGIN
password=PASSWORD
baseurl=http://download.rudder.io/rpm/9.0/RHEL_7/
gpgcheck=1
gpgkey=https://repository.rudder.io/rudder_release_key.pub' > /etc/yum.repos.d/rudder.repo

----


Install the package:

----

yum install rudder-agent

----
