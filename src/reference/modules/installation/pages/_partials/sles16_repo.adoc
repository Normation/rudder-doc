* on SLES 16:

----

zypper ar -n 'Rudder 9.1' http://repository.rudder.io/rpm/9.1/SLES_16/ Rudder

----

[NOTE]
====

Rudder server and relay does not support running on a SLES 16 system with SELinux in enforce mode for now.

====

[NOTE]
====

If you have an active subscription, use the following to get access to long term support (you need to replace
the username and the password by your Rudder account):

----

zypper ar -n 'Rudder 9.1' https://LOGIN:PASSWORD@download.rudder.io/rpm/9.1/SLES_16/ Rudder

----

====

Update your local package database to retrieve the list of packages available in our repository:

----

zypper refresh

----
