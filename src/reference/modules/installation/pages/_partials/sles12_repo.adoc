zypper ar -n 'Rudder 7.1' http://repository.rudder.io/rpm/7.1/SLES_12/ Rudder

[NOTE]
====

If you have an active subscription, use the following to get access to long term support (you need to replace
the user name and the password by your Rudder account):

----

zypper ar -n 'Rudder 7.1' https://LOGIN:PASSWORD@download.rudder.io/rpm/7.1/SLES_12/ Rudder

----

====

Update your local package database to retrieve the list of packages available on our repository:

----

zypper refresh
