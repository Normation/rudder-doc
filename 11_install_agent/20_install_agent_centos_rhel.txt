=== Install Rudder Agent on RedHat or CentOS

Download the package applicable to your version of RedHat/CentOS and to its
architecture on 

----

http://www.rudder-project.org/rpm-2.6/RHEL_5/
http://www.rudder-project.org/rpm-2.6/RHEL_6/

----

Or you can define a yum repository for RedHat/CentOS 6:

----

$ echo "[Rudder_2.6]
name=Rudder 2.6 Repository
baseurl=http://www.rudder-project.org/rpm-2.6/RHEL_6/
gpgcheck=0
" > /etc/yum.repos.d/rudder.repo

----

Or for RedHat/CentOS 5:

----

$ echo "[Rudder_2.6]
name=Rudder 2.6 Repository
baseurl=http://www.rudder-project.org/rpm-2.6/RHEL_5/
gpgcheck=0
" > /etc/yum.repos.d/rudder.repo

----


Install the package:

----

rpm -Uhv rudder-agent-2.6.0-1.EL.5.x86_64.rpm

----

Or if a yum repository has been set:

----

yum install rudder-agent

----
